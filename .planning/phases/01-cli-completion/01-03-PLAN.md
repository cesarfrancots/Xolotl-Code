<objective>
Wire `--resume <id>` (no trailing slash command) to open the interactive REPL with the loaded session instead of printing a message and exiting.

Currently `CliAction::ResumeSession { command: None }` just prints "Restored session from X" and returns. The fix: when `command` is `None`, load the session into a fresh `LiveCli` and call `run_repl()` (or an equivalent that accepts an already-built `LiveCli`).

The simplest correct path reuses `LiveCli::load_session()` which already fully rebuilds `self.runtime` with the loaded session. The change is:
1. Refactor `run_repl()` to accept `Option<(Session, PathBuf)>` for an initial session, OR
2. Start a fresh `LiveCli`, immediately call `cli.load_session(id)` before entering the REPL loop.

Approach 2 is preferred — it involves zero new abstractions and reuses the tested `load_session()` path that already handles path resolution, session loading, and runtime rebuild.

Purpose: CLI-04 is a core usability requirement — without it, users lose prior context on every launch.
Output: `main.rs` with targeted changes to `run()` match arm and `run_repl()` signature; tests green.
</objective>

  **Step 1 — Add `run_repl_resumed()` function:**

  Add a new function near `run_repl()`:
  ```rust
  fn run_repl_resumed(
      session_arg: &Path,
      model: String,
      auto_accept: bool,
      budget: Option<f64>,
  ) -> Result<(), Box<dyn std::error::Error>> {
      // Resolve path: if session_arg is a bare name (no .json), look it up in sessions_dir.
      // If session_arg already points to an absolute path or a .json file, use it directly.
      let resolved_path = {
          let s = session_arg.to_string_lossy();
          let sessions = sessions_dir();
          if session_arg.is_absolute() {
              session_arg.to_path_buf()
          } else if s.ends_with(".json") {
              sessions.join(session_arg)
          } else {
              sessions.join(format!("{s}.json"))
          }
      };

      if !resolved_path.exists() {
          return Err(format!("session not found: {}", resolved_path.display()).into());
      }

      let mut cli = LiveCli::new(model, true, auto_accept)?;
      if let Some(b) = budget {
          cli.set_budget(b);
      }
      // Load the session into the fresh LiveCli (rebuilds runtime with session history)
      let session = runtime::Session::load_from_path(&resolved_path)?;
      let msg_count = session.messages.len();
      cli.auto_save_path = resolved_path.clone();
      cli.cache_scope = build_prompt_cache_scope_for_session_path(&cli.auto_save_path);
      cli.runtime = build_runtime(
          session,
          cli.model.clone(),
          cli.system_prompt.clone(),
          Arc::clone(&cli.mcp),
          true,
          &cli.cache_scope,
      )?;
      style::print_ok(&format!(
          "Resumed session → {} ({} messages)",
          style::shorten_path(&resolved_path),
          msg_count
      ));

      // Enter the REPL loop (same as run_repl but with pre-loaded session)
      let mut editor = input::LineEditor::new("› ");
      print_startup_banner(&cli);
      // Delegate to the shared REPL loop body
      run_repl_loop(cli, editor)
  }
  ```
