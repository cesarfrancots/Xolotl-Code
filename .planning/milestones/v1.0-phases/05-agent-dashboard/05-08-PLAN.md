In `WorktreeManager::add()` (worktree.rs lines 62-88), after the `git worktree add` fails, check if stderr contains "already exists" and if so, run `git branch -D <branch>` to remove the stale branch, then retry the worktree add. Only retry once — if it fails again, return the error unchanged.

Replace the current error block (lines 80-83):
```rust
if !output.status.success() {
    let stderr = String::from_utf8_lossy(&output.stderr).into_owned();
    return Err(WorktreeError::GitFailed(stderr));
}
```
with:
```rust
if !output.status.success() {
    let stderr = String::from_utf8_lossy(&output.stderr).into_owned();
    // If the branch already exists (stale from a prior crashed spawn), delete it and retry.
    if stderr.contains("already exists") {
        let del = std::process::Command::new("git")
            .args(["branch", "-D", branch])
            .current_dir(&self.repo_root)
            .output()?;
        if del.status.success() {
            // Retry the worktree add with the now-deleted branch.
            let retry = std::process::Command::new("git")
                .args(["worktree", "add", "-b", branch, path_str])
                .current_dir(&self.repo_root)
                .output()?;
            if !retry.status.success() {
                let retry_stderr = String::from_utf8_lossy(&retry.stderr).into_owned();
                return Err(WorktreeError::GitFailed(retry_stderr));
            }
            let mut active = self.active.lock().unwrap();
            active.insert(agent_id.clone(), path.clone());
            return Ok(path);
        }
    }
    return Err(WorktreeError::GitFailed(stderr));
}
```

**Part B — worktree.rs: also delete the branch on worktree remove**

In `WorktreeManager::remove()` (lines 97-125), after the `git worktree remove --force` succeeds, also delete the branch so it cannot cause future collisions. Add after the success check:
```rust
// Delete the branch that was created exclusively for this worktree.
// Failure is non-fatal — prune() at next startup handles any residue.

    .args(["branch", "-D", &format!("agent/{}", agent_id)])
    .current_dir(&self.repo_root)
    .output();
```

Wait — the branch name is not stored in WorktreeManager (only the path is). The branch name is passed to `add()` but not kept. Instead, look up the branch from the worktree path using `git worktree list --porcelain` and parse the branch line, OR store the branch alongside the path.

Update `add()` to store `(path, branch.to_string())`, update `remove()` to retrieve and delete the branch, update `list()` and `get_path()` to destructure the tuple.

All callers of `list()` only use the path portion already. `get_path()` only returns the path. No external API changes needed beyond the internal tuple.

**Step 1 — Track cumulative cost unconditionally**

Replace:
```rust
if let AgentEvent::TurnCompleted { ref usage } = event {
    if let Some(budget) = handle.budget_dollars {
        let new_cost = handle.accumulate_cost(usage, &handle.model);
        if new_cost >= budget {
            ...
        }
    }
}
```
with:
```rust
if let AgentEvent::TurnCompleted { ref usage } = event {
    let new_cost = handle.accumulate_cost(usage, &handle.model);
    if let Some(budget) = handle.budget_dollars {
        if new_cost >= budget {
            let _ = handle.event_tx.send(
                AgentEvent::StateChanged(AgentState::Failed)
            ).await;
            let _ = handle.event_tx.send(AgentEvent::Error {
                message: format!("Budget exceeded: ${:.4}", new_cost),
            }).await;
        }
    }
}
```

**Step 2 — Fire notification on Done/Failed**

Add a notification block immediately after `let _ = app_handle.emit(&channel, &event);` and the TurnCompleted block. Insert before the `Err(RecvError::Lagged...)` match arm:

```rust
// OS notification on terminal states (D-12, D-13, D-14).
if let AgentEvent::StateChanged(ref state) = event {
    if matches!(state, AgentState::Done | AgentState::Failed) {
        let cost = handle.cumulative_cost
            .lock()
            .map(|g| *g)
            .unwrap_or(0.0);
        let state_label = if matches!(state, AgentState::Done) { "Done" } else { "Failed" };
        let title: String = handle.task.chars().take(60).collect();
        let body = format!("{} — ${:.4}", state_label, cost);
        let _ = app_handle
            .notification()
            .builder()
            .title(&title)
            .body(&body)
            .show();
    }
}
```
