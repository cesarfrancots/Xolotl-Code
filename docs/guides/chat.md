# Chat

> **Status:** outline — expand with screenshots once the website build is set up.

The chat pane is the primary surface of the desktop app.

## Covered here (to expand)

- **Streaming responses** — tokens render live as they arrive.
- **Separate chain-of-thought** — reasoning models (Kimi, DeepSeek) stream `reasoning_content` first, then the answer; both are shown in distinct tracks. See [Providers & models](../providers.md#reasoning-models).
- **Tool-use display** — tool calls and results are surfaced inline.
- **Inline diffs** — file edits show as diffs you can review.
- **Command palette** — quick actions and model switching.
